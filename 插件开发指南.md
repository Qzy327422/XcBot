# XcBot 插件开发指南

## 目录结构

```
plugins/
└── your_plugin/
    ├── setup.py      # 或 main.py，插件入口
    └── metadata.yaml # 插件元数据
```

## 入口文件

入口文件必须定义以下两个内容：

```python
# 触发关键词，使用 "Any" 表示响应所有消息
TRIGGHT_KEYWORD = "关键词"

async def on_message(user_message, user_id, group_id, actions, is_group):
    # user_message: 消息内容
    # user_id: 发送者 QQ 号
    # group_id: 群号（私聊为 None）
    # actions: 发送消息等操作对象
    # is_group: 是否群聊
    await actions.send(
        group_id=group_id,
        message=...
    )
    return True  # 返回 True 表示已处理，阻止后续插件执行
```

## metadata.yaml

```yaml
name: your_plugin
version: 1.0.0
description: 插件描述
author: 你的名字
entry: setup.py
```

## 向商店提交插件

### 方式一：GitHub 网页上传（推荐新手）

1. 打开 [Qzy327422/XcBot-Plugins](https://github.com/Qzy327422/XcBot-Plugins)，点击右上角 **Fork**
2. 进入你 Fork 后的仓库，点击 `plugins/` 文件夹
3. 点击右上角 **Add file → Upload files**，把你的插件文件夹打包成 zip 后上传，或者用 **Create new file** 逐个创建文件
4. 填写提交信息，点击 **Commit changes**
5. 回到仓库主页，点击 **Contribute → Open pull request**，描述插件功能后提交

### 方式二：git 命令行

```bash
git clone https://github.com/你的用户名/XcBot-Plugins.git
cd XcBot-Plugins/plugins
mkdir your_plugin && cd your_plugin
# 放入 setup.py 和 metadata.yaml
cd ../..
git add .
git commit -m "Add your_plugin"
git push
# 然后在 GitHub 网页提交 Pull Request
```

## 本地安装插件

在 WebUI 插件商店页面右下角点击 **⇧** 上传按钮，选择打包好的 `.zip` 文件即可。

zip 文件结构：

```
your_plugin.zip
├── your_plugin/      # 顶层目录（可有可无，会自动剥离）
│   ├── setup.py
│   └── metadata.yaml
```

安装后如插件未自动生效，发送 `/重载插件` 命令。
